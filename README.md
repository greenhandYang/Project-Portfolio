# Project-Portfolio
作品名称：诗檐明烛
作品简介：随着人工智能技术的不断进步，传统文化渐渐淡出人们的视野，其中包括了中国古诗词的传承与传播。
然而，借助人工智能技术来赋予中国传统文化新的生命，可以更加方便地将其传扬开来。尽管中国古诗词数量庞大，
但是一个人的精力是有限的，很难深入了解其中的精华。基于这一认识，我们开发了一款名为“诗檐明烛”的应用程序。
该应用不仅能够为用户提供智能推荐的古诗词，还拥有诗词聊天问答机器人，帮助用户更方便地了解诗词文化。
通过这样的方式，我们既能够帮助用户更好地了解中华诗词文化，也能够更好地传扬和宣扬这一宝贵的文化遗产。
本项目曾荣获2024年中国大学生计算机设计大赛省级一等奖。由于项目依赖的云服务器租期已满，目前暂无法提供在线演示环境或完整可运行源码。
但为了完整展示项目成果与设计思路，本仓库提供了：
1、全程功能演示视频：涵盖核心业务流程与亮点功能
2、完整的技术文档：包含系统架构设计、数据库设计、系统关键界面截图、主要技术简介、算法介绍及测试报告
